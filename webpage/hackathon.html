The Problem

Cloud Dependency

Require constant internet connectivity and expose user data to third parties. Your conversations, prompts, and personal information leave your device.

💸

Payment Barriers

Can't autonomously access paid APIs without breaking agent autonomy. Every transaction requires manual human intervention.

🔐

User Control

Either too restrictive (no autonomy) or too permissive (security risks). There's no middle ground for safe, autonomous operations.

Try It Yourself

Experience the future of AI agent payments today

📱 Download SolAIBot APK 💻 Clone Agent Kit

Quick Start:

SolAIBot (Android):

  1. Download APK and install on Android 12+
  2. Install Solflare or MWA-compatible wallet
  3. Let the app download Hammer 2.1 model (~1.3GB)
  4. Ask AI to access paid content and approve payments

Agent Kit (Node.js):

  1. Clone repository and run pnpm install
  2. Build with pnpm build
  3. Configure wallet in examples/.env
  4. Run CLI demo or MCP server
